# Toy Input Transcription Data Set

This exemplary transcription data are the specific deregulated gene
expression of the 786-O cell line from the NCI60 dataset.

## Usage

``` r
data(toy_RNA)
```

## Format

An object of class “`numeric`” containing the t-values of 9300 genes,
which are named with gene symboles matching the toy network.

## Source
