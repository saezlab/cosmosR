# Toy Metabolic Input Data

This metabolic data are a subset from the metabolic measurements of the
786-O cell line from the NCI60 dataset.

## Usage

``` r
data(toy_metabolic_input)
```

## Format

An object of class “`numeric`” containing the t-values of 2 metabolites,
which are named with metabolite HMDB Ids matching the toy network.

## Source

Subset of:
<https://github.com/saezlab/COSMOS_MSB/blob/main/data/metab_input_COSMOS.csv>

## References

Dugourd, A., Kuppe, C. and Sciacovelli, M. et. al. (2021) *Molecular
Systems Biology*. **17**, e9730.

## Examples

``` r
data(toy_metabolic_input)
```
